Expense and Profit Analyser

A simple web application to analyse expenses and sales data and calculate profit or loss.
This project focuses on basic financial analysis with a clean and easy-to-use interface.

Features
- Upload expense and sales data
- Calculate total expenses, total sales, and profit
- Display results using charts
- Simple and clean UI

Tech Stack
- Frontend : React (Vite)
- Backend  : Python (FastAPI)
- DataBase   : postgresql

Project Structure
- backend   -> API and business logic
- frontend  -> User interface

How It Works
- User uploads expense and sales files
- Backend processes the data
- Application calculates totals and profit
- Results are shown in text and charts

Purpose
- This project was built to improve full-stack development skills and to understand real-world
data flow between frontend and backend.
